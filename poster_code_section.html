capture.pyPi camera capture — locks white balance & focus, saves timestamped JPEG.
process.pyUndistorts image → detects ArUco marker for px/mm scale → extracts tool silhouette → expands outline by offset (mm) for foam clearance.
vectorize.pyConverts binary mask to SVG vector path via vtracer.
export.pyPatches SVG units to real-world mm, sets viewBox, converts to hairline strokes for laser cutter.
config.pyAll tunable parameters — calibration matrix, crop bounds, marker size, outline offset, output paths.

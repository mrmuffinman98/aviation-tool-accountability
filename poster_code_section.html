capture.py Step 1
  • Triggers Raspberry Pi Camera Module via Picamera2
  • Locks white balance & manual focus for repeatable shots
  • Saves timestamped JPEG to images/ folder
process.py Step 2 — Core Pipeline
  • Undistort — corrects lens distortion from calibration matrix
  • Crop — trims to light board interior, removes edge artifacts
  • Scale detection — locates ArUco marker → computes px/mm ratio
  • Silhouette — thresholds dark tool vs. bright background; expands outline by configured offset (mm) for foam clearance
# Scale from physical marker size pixels_per_mm = avg_side_px / ARUCO_MARKER_SIZE_MM # Expand outline for foam clearance offset_px = round(OUTLINE_OFFSET_MM * pixels_per_mm) mask = cv2.dilate(mask, kernel(offset_px))
vectorize.py Step 3
  • Crops mask to tool bounding box before tracing
  • Passes binary image to vtracer (Rust-based tracer)
  • Returns clean SVG path string
export.py Step 4
  • Converts SVG pixel units → real-world millimeters using px/mm ratio
  • Sets explicit viewBox so Illustrator renders at correct scale
  • Strips border rectangle; converts fills to hairline strokes for laser
width_mm = width_px / pixels_per_mm height_mm = height_px / pixels_per_mm root.set("width", f"{width_mm:.4f}mm") root.set("viewBox", f"0 0 {width_px} {height_px}")
config.py Settings
All tunable parameters in one file — no magic numbers in pipeline code.
  • Camera calibration matrix & distortion coefficients
  • Capture resolution, crop bounds, threshold value
  • ArUco marker size (mm), outline offset (mm)
  • Output paths for images and SVG files
